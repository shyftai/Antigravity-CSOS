# Customer Segments

Segment definitions determine service levels, touchpoint cadence, and resource allocation.

## Segment definitions

### Enterprise
- **ARR threshold:** $100K+
- **Employee count:** 500+
- **CSM ratio:** 1:10-15
- **Touchpoint cadence:** Weekly check-in, monthly strategic review, quarterly QBR
- **Onboarding:** White-glove, dedicated onboarding manager, 90-day program
- **Support tier:** Priority (4h SLA)
- **Executive sponsor:** Yes (VP+)
- **Success plan:** Required, reviewed monthly
- **Expansion motion:** Proactive, CSM-driven

### Mid-Market
- **ARR threshold:** $25K-$99K
- **Employee count:** 100-499
- **CSM ratio:** 1:25-40
- **Touchpoint cadence:** Bi-weekly check-in, quarterly QBR
- **Onboarding:** Guided, shared onboarding resources, 60-day program
- **Support tier:** Standard (8h SLA)
- **Executive sponsor:** Optional (Director+)
- **Success plan:** Recommended, reviewed quarterly
- **Expansion motion:** Signal-based, CSM identifies triggers

### SMB
- **ARR threshold:** $5K-$24K
- **Employee count:** 10-99
- **CSM ratio:** 1:80-120 (pooled or tech-touch)
- **Touchpoint cadence:** Monthly automated check-in, bi-annual review
- **Onboarding:** Self-serve with guided resources, 30-day program
- **Support tier:** Standard (24h SLA)
- **Executive sponsor:** No
- **Success plan:** Template-driven, self-managed
- **Expansion motion:** Product-led, automated triggers

### Strategic
- **ARR threshold:** $250K+ or logo value
- **Employee count:** Any (qualification is strategic, not size)
- **CSM ratio:** 1:3-5
- **Touchpoint cadence:** Weekly strategic call, monthly exec alignment, quarterly business review
- **Onboarding:** Custom program, dedicated project manager, 120-day program
- **Support tier:** Premium (1h SLA, dedicated support engineer)
- **Executive sponsor:** Required (C-level)
- **Success plan:** Custom, reviewed bi-weekly
- **Expansion motion:** Account-planned, multi-threaded
